Using CIFcheck from CRYSTALS

The IUCr CIFcheck server provides several levels of checking

  1. General version of checkCIF
    http://checkcif.iucr.org
    This is the site we recommend non-IUCr journals to use for structural checking, and for general checking.
  2. IUCr Journal specific version of checkCIF - Acta Crystallographica Sections C and E
    http://journals.iucr.org/services/cif/checking/checkfull.html
    This site runs all the general tests as well as specific tests to check that the CIF includes e.g. an Abstract for publication.
  3. IUCr Journal specific version of checkCIF - all other IUCr journals
    http://journals.iucr.org/services/cif/checking/checkbasic.html
    This is the site we recommend authors to use for our other journals. It should provide the same results as site (1).
  4. These other URLs should not be included in your documentation as they are now depreciated.
    journals.iucr.org/services/cif/checking/checkform
    journals.iucr.org/services/cif/checkcif
    ww1.iucr.org/cif/checkcif (A list of current choices)
