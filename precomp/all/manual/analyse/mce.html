Contour Levels

MCE contour viewer can be used to visualise both the electron density within the slant slab, and the volume outside of the Van der Waals surface. The electron density colours and levels are defined by the Primary Map, the surface properties by the Secondary Map.
To save settings so that they can be re-used:
1. Load some map.
2. In "Preferences" "Display style" window check "Use previous map settings".
3. In "Preferences" "Colour Setting" set map iso-levels and background colour. check "Save Steetings"
4. menu->edit->Save map settings.
5. Load another map. This map starts with saved iso-levels.

For disabling this possibility, uncheck "Use previous map settings"
